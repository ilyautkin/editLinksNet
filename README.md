editLinksNet
============
Version: 0.1.0 Author: Ilya Utkin <ilyautkin@mail.ru>
------------

Helps to make links on site pages.

------------

IMPORTANT: TAG [[*content]] IN YOUR TEMPLATES
HAVE TO BE WRAPPED BY DIV WITH ID "CONTENT"!

Example:

<h1>[[*pagetitle]]</h1>
<div id="content">
  [[*content]]
</div>
